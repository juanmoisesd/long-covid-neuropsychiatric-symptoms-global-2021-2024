## Overview

Global and country-level statistics on **neurological and psychiatric symptoms in Long COVID** (Post-Acute Sequelae of SARS-CoV-2, PASC) from systematic reviews and cohort studies (2021-2024).

## Key Prevalence Data

| Symptom | Global Range | Country Example |
|---|---|---|
| Fatigue | **74%** | Bangladesh: 82.9% → 16.1% (week 31) |
| Brain fog / Cognitive impairment | **32–60%** | Colombia: 60% at 2–3 years |
| Sleep disorders | **41–64%** | US non-hospitalized: 64% |
| Post-exertional malaise | **58.7%** | Brazil FIOCRUZ cohort |
| Anxiety | **23–44%** | Colombia: 44% |
| Depression | **12–30%** | Global diagnostic criteria |
| Cognitive impairment | **17.5–33%** | OECD: 16–33% |
| Muscle pain | **42%** | Global cohorts |
| Numbness/Tingling | **41%** | Global cohorts |

**1/6 to 1/3** of infected individuals in OECD countries report persistent cognitive symptoms.
